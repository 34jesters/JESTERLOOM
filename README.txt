JesterLoom

3D conversation tree explorer for branching AI chats. No setup, just works.

═══════════════════════════════════════════════════════════════════

WHAT IS THIS?

Talk to AI, explore different conversation paths, see it all in 3D. vibe coded in a day because other looms didnt work for me.

═══════════════════════════════════════════════════════════════════

FEATURES

• Just open the HTML file - no bullshit
• Works with any OpenRouter model - Claude, GPT, Llama, whatever
• 3D tree visualization - actually see where your conversations branch
• Generate multiple responses - get 1, 3, or 5 options per prompt
• Save/load trees - keep your explorations

i am so not responsible if this somehow explodes your shit--i am literally just now figuring all of this out

═══════════════════════════════════════════════════════════════════

SETUP

1. Download jesterloom.html
2. Open it in your browser
3. Put in your OpenRouter API key (get one at openrouter.ai)
4. Start talking

═══════════════════════════════════════════════════════════════════

USAGE

Type a prompt, hit generate, click on nodes to explore branches. That's it.

• Drag to rotate the 3D view
• Scroll to zoom
• Click nodes to select them
• Generate from any node to create new branches

═══════════════════════════════════════════════════════════════════

WHY?

other loom architectures have heavy dependencies and were difficult to launch.

JesterLoom fixes that. Same concept, easier to use, works with any(?) openrouter model.
